    CalibrationAPI

    Extends:

    events~EventEmitter → CalibrationAPI

    API para calibração dos modelos de regressão disponiveis no ICan.js Esta API captura a posição do mouse do usuário assim como a posição do nariz, para que seja possível para os modelos de regressão mapear a relação entre as duas variáveis.
