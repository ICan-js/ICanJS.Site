    public calibrate(regressionModel: Regression, fncFinish: Function, fncUpdate: Function, calibrationOptions: Object) source 

    import {calibrate} from 'ican.js/src/common/Calibration/index.js'

    Facade para a API de calibração de modelo de regressão que será utilizado nos métodos de controle do mouse com o nariz. Aqui básicamente são captados uma quantidade X (Por padrão 15) de pontos pontos do mouse do usuário e também da posição de seu nariz e então com estes dados criar um modelo de regressão (Definido pelo usuário) que será devolvido através da função de callback.

    Params:

    NameTypeAttributeDescription
    regressionModel Regression

    Modelo de regressão que deverá ser utilizado no projeto
    fncFinish Function

    Função que será utilizada para entragar o modelo de regressão calibrado
    fncUpdate Function

    Função que será utilizada sempre que um ponto for atualizado
    calibrationOptions Object

    Opções que podem mudar os objetos do CANVAS de calibração
    calibrationOptions.pointSize Number

    Tamanho das elipses a serem exibidas na calibração
    calibrationOptions.pointsToStorage Number

    Quantidade de pontos que devem ser armazenados de cada elipse

    NameTypeAttributeDescription

    Array com um conjunto de saídas do método predictFrame da classe MobileNetV1Libras

    Return:

    String

    Palavra que o usuário está descrevendo através de gestos

    public librasWriter(videoCapture: HTMLMediaElement, delay: Number, nFrames: Number, fnc: Function) source 

    import {librasWriter} from 'ican.js/src/common/Writer/index.js'

    Função para escrita de textos com gestos de Libras na câmera. Para a utilização desta função será necessário definir o delay em que cada frame é capturado e também a quantidade de frames que deverão ser utilizados para definir o texto escrito pelo usuário. Isto por que esta função captura uma quantidade N de imagens, em X delay e então cria uma média de resultados de predição, assim a palavra com a maior média é utilizada como verdade ao o que o usuário estava dizendo nos gestos.

    Params:

    NameTypeAttributeDescription
    videoCapture HTMLMediaElement

    Stream da webcam do usuário
    delay Number

    Tempo de espera entre cada gesto em segundos
    nFrames Number

    Quantidade de frames a serem captados para o cálculo da média
    fnc Function

    Função que será utilizada para devolver os resultados calculados

    public screenScroller(regressionModel: Regression, filter: String) source 

    import {screenScroller} from 'ican.js/src/common/Viewer/index.js'

    Facade para a utilização do PoseNet em Screen Scrollers (Controle da posição na tela utilizando gestos com a cabeça). A metodologia utilizada aqui foi apresentada no WebGazer.js

    Params:

    NameTypeAttributeDescription
    regressionModel Regression

    Modelo de regressão que será utilizado junto a rede neural para criar o controle do mouse do usuário
    filter String

    Filtro que pode ser aplicado as localizações onde está a div (Que representa o ponteiro do mouse) está sendo colocada. Você pode utilizar os filtros "median" ou "mean". Por padrão nenhum filtro é aplicado

    See:

    • https://webgazer.cs.brown.edu/
